# Mermaid Linter

## Usage
Run the script to validate a specific Mermaid JSON file generated by the topology-mapper.

```bash
python3 .opencode/skills/mermaid-linter/run.py --file DocumentationFactory/output/artifacts/architecture-diagrams.json
```

If syntax errors are found, the script will exit with status `1` and output the exact line number of the invalid syntax.
